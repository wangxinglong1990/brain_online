# 脑机接口技术官网

基于React开发的脑机接口技术展示网站，展示真实的MEG脑信号数据可视化。

## 技术栈

- React 18
- Framer Motion (动画效果)
- React Intersection Observer (滚动检测)
- Plotly (数据可视化)

## 功能特点

- 🧠 真实MEG脑信号数据展示
- 📊 实时数据可视化
- 🎨 现代化UI设计
- 📱 响应式布局
- ⚡ 流畅的动画效果

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建生产版本
npm run build
```

## 部署

项目已配置GitHub Pages自动部署：

```bash
npm run deploy
```

## 数据来源

MEG脑信号数据来自Gwilliams2022数据集，包含208个传感器通道的真实临床记录。
